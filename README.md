## 使用方法：

### （如果你直接使用代码）找到main.py运行，分别输入：

-  	完整的存储路径文件夹 
-  	想要爬的那位用户的推特ID（@后文字） 。
-  	页面滚动次数（取决于你想爬的用户有多少图来自行决定，建议50次）
-  	还有Edge WebDriver的完整路径（包括.exe）
